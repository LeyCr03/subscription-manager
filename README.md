## About the project

Subscription Management web app with [Next.js](https://nextjs.org/learn), [TanstackQuerry](https://tanstack-com.translate.goog/query) and [Shadcn](https://ui.shadcn.com)

## Some Features:
- Auth screens
- Dashboard
- Metrics

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
